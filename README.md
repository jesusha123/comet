# Comet

Comet is an API client for sending HTTP requests

## Features

- Fast
- Native application
- Low memory footprint
- Cross-platform
- Open source
- Local only / no phoning home

## Build

- Install Qt 6 SDK
- Install CMake
- Generate Makefile
```sh
cmake .
```
- Make application
```sh
make
```
- Run application
```sh
# macOS
./comet.app/Contents/MacOS/comet
```
